# 1. INTRODUCTION

This laboratory manual is designed in accordance with the **CSIT-606 Android Programming syllabus** under the AICTE Flexible Curricula for VI Semester.

The experiments follow a progressive structure — from environment setup and lifecycle fundamentals to modern Android practices using **Room, RecyclerView, WorkManager, and Storage Access Framework**.

Unlike older Android labs, this manual uses **current industry-standard APIs** and avoids deprecated approaches like `AsyncTask`, `ListView`, and raw `SQLiteOpenHelper`.

---

# 2. COURSE OBJECTIVES

- To set up and configure the Android development environment
- To understand the Android **Activity Lifecycle**
- To design responsive UIs using **ConstraintLayout** and Material themes
- To manage data with **Room** persistence library
- To execute background tasks using **WorkManager**
- To handle files with **App-Specific Storage and SAF**

---

# 3. SOFTWARE REQUIREMENTS

- **Android Studio** (Hedgehog or later)
- **JDK 17**
- **Android SDK** (API Level 26+ recommended, API 34 target)
- **Android Virtual Device (AVD)** or a physical Android device (Android 8.0+)
