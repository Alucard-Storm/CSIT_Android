# CSIT-606 Android Programming – Official Syllabus

**University:** Rajiv Gandhi Proudyogiki Vishwavidyalaya (RGPV), Bhopal  
**Program:** B.E. Computer Science & Information Technology  
**Semester:** VI | **Scheme:** AICTE Flexible Curricula

---

## Course Objectives

- To install and configure Android application development tools
- To design and develop user interfaces for the Android platform
- To save state information across important operating system events
- To apply Java programming concepts to Android application development

---

## Course Outcomes

1. Experiment on Integrated Development Environment for Android Application Development
2. Design and Implement User Interfaces and Layouts of Android App
3. Use Intents for activity and broadcasting data in Android App
4. Design and Implement Database Application and Content Providers
5. Experiment with Camera and Location Based service and develop Android App with Security features

---

## Course Contents

### UNIT I – Introduction to Android
- Background about mobile technologies
- Overview of Android and Android Architecture
- Open Handset Alliance
- Android Internals: ART vs Dalvik
- Why use Android for mobile development

### UNIT II – Android Development Environment
- Android Studio setup and project structure
- Android SDK, Gradle build system, AVD Manager
- AndroidManifest.xml and R.java

### UNIT III – Activity, Lifecycle & Components
- Four Android Application Components
- Activity Lifecycle: all 7 callbacks
- Saving and restoring state

### UNIT IV – Intents, UI Design & Layouts
- Explicit and Implicit Intents
- Passing data between activities
- Layout types: ConstraintLayout, LinearLayout, ScrollView
- Material Design components and themes
- Density units: dp, sp, px

### UNIT V – Modern Android APIs
- RecyclerView and ViewHolder pattern
- Room Persistence Library
- WorkManager for background tasks
- Scoped Storage and Storage Access Framework
